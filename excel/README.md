# 📊 Excel Projects

Aqui é onde estão os projetos realizados utilizando a ferramenta do Excel. Foco em organização, tratamento de dados, criação de ferramentas administrativas funcionais e dashboards.

---

### 🛠️ O que você encontrará aqui:
* **Dashboards Interativos:** Design de interface (UI/UX) focado em tomada de decisão.
* **Power Query (ETL):** Limpeza, tratamento e modelagem de dados complexos com Linguagem M.
* **Análise Dinâmica:** Tabelas e Gráficos Dinâmicos com Segmentadores (Slicers) sincronizados.
* **Fórmulas Avançadas:** Lógica condicional, busca (XLOOKUP/VLOOKUP) e funções estatísticas.

---

### 📂 Projetos em Destaque:

#### 1. 🎮 Steam 2024: Market Trends & Performance
**Foco:** Análise de Mercado e Comportamento do Consumidor.
Este dashboard analisa o faturamento e a satisfação dos usuários na plataforma Steam durante o ano de 2024. 
* **O Desafio:** Tratar uma base de dados volumosa com valores nulos de avaliações e concentrar o faturamento de milhares de títulos em rankings legíveis.
* **Destaque:** Implementação de análise de "Quebra de Expectativa", provando que jogos com menor orçamento (Indie) possuem avaliações superiores aos grandes títulos (AAA), apesar da diferença de faturamento.
* **Status:** Finalizado 🟢

#### 2. 🛒 Quick Commerce Dashboard
**Foco:** Inteligência de Vendas e Comportamento de Consumo.
Este projeto consistiu na estruturação e análise de uma base de dados de vendas rápidas, focando na identificação de padrões de faturamento e métodos de pagamento.

* **O Desafio:** Organizar dados brutos de transações para extrair métricas claras de performance, como Ticket Médio e volume de pedidos por categoria.

* **Destaque:** Uso do Power Query para o tratamento inicial da base, garantindo que a análise refletisse a realidade das operações de varejo de forma organizada e visual.

* **Status:** Finalizado 🟢
